# Minimal Skill Example

A minimal example skill showing the simplest possible structure.

## When to Use

- Use this as a reference for creating simple skills
- This skill demonstrates the minimum required files (just SKILL.md)

## Instructions

1. Copy this skill to your project or global skills directory (e.g. `.cursor/skills/` or `.claude/skills/`, depending on your client)
2. Rename the folder to match your skill name
3. Update the `name` and `description` in the frontmatter
4. Replace these instructions with your own

## Notes

- Only `SKILL.md` is required for a valid skill
- Add `scripts/`, `references/`, and `assets/` directories as needed
- See the `TEMPLATE` skill for a complete example with all optional directories
